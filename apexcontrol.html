About Apex Control

Apex Control is a powerful mobile application that provides users with advanced control over their device's performance and settings. With an intuitive interface, users can easily customize various aspects of their device to suit their preferences, ensuring an optimized user experience.

Key Features

  • Advance Display settings
  • Kernel settings
  • Advance Device info
  • various Reboot options
  • Advance System settings
  • Rom backup with Critical partitions
  • Device Optimization & Cleaning
  • Adding more features in upcoming updates

How to Use

  1. Open and Grant root permission.
  2. Explore the various sections and features.
  3. Follow given instructions.
  4. Roaming button is for MYUi CN rom specific.
  5. Have any question? Feel free to ask in the support group
